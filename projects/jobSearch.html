Software Developer Job Search

Previous Next

The Software Developers Job Search application was designed for developers out of school who are looking for jobs. In addition to job listings, the application also provides news articles about the company so developers can make a more informed decision about where they work. This application utilizes GitHub Jobs API, GNews API, Zip Code API, Fomantic UI, Lodash, and JQuery.

GitHub Repository: https://github.com/thusted/P1_JobSearch

Deployed Application: https://thusted.github.io/P1_JobSearch/
